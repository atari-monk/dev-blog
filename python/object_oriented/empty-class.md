# Empty Class

- Minimal empty class definition:
  ```python
  class EmptyClass:
      pass
  ```
- `pass` acts as a syntactic placeholder

- Empty class with docstring:
  ```python
  class EmptyClass:
      """An empty class with no attributes or methods."""
      pass
  ```

- Both variants:
  - Can be instantiated
  - Can serve as base classes
  - Are valid Python syntax
