Here's an empty Python class:

```python
class EmptyClass:
    pass
```

This is the simplest possible class definition in Python. The `pass` statement is used as a placeholder where code is required syntactically but you don't want any action to occur.

You can also create a class with a docstring (which is recommended for better documentation):

```python
class EmptyClass:
    """An empty class with no attributes or methods."""
    pass
```

Both versions are valid empty classes that can be instantiated and used as base classes for inheritance.
